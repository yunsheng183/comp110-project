# COMP110 Data Analysis Project

## Idea
We analyzed the idea that making lectures more engaging with real-world examples improves student understanding.

---

## Analysis Summary
We used survey data to explore student engagement and learning experience.

We:
- Loaded dataset using read_csv_rows
- Selected relevant columns using select
- Counted responses using count
- Created multiple visualizations using seaborn


## Visualizations

My visualization shows the distribution of student majors in the dataset. 

We can observe that a few majors have significantly higher counts compared to others, indicating that the dataset is not evenly distributed across all academic disciplines.

This suggests that certain majors may be more represented in the survey, which could potentially influence the overall results of the analysis.

## Conclusion
The analysis shows that student engagement is related to clearer and more relatable lecture content.

This suggests that adding real-world examples may improve understanding.

However, limitations include self-reported survey bias and limited sample size.
